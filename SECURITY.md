# Security

## Reporting a vulnerability

If you find a security issue in the API or in this repo's example code, email `security@pytheum.com` with details. We respond within 48 hours.

Please don't open public GitHub issues for security.

## Scope

This repo is docs + examples; the API itself lives at `api.pytheum.com`. Both are in scope. Third-party services we link to (Pinecone, Supabase, Vercel, etc.) are not — report those to their respective vendors.
